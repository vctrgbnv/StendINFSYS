# Stend INFSYS

Учебный проект для сбора и визуализации параметров винтомоторных групп. Стек: Django, PostgreSQL, InfluxDB v2, Chart.js.

## Быстрый старт (локально)
1. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python3 -m venv .venv
   . .venv/bin/activate
   pip install -r requirements.txt
   ```
2. Скопируйте `.env.example` в `.env` и при необходимости скорректируйте переменные.
3. Поднимите базы:
   ```bash
   docker compose up -d
   ```
4. Выполните миграции и создайте суперпользователя (будет дописано по мере разработки).

Подробнее о функциональности и импорте CSV будет добавлено в следующих коммитах.
